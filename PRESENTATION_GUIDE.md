# 课堂展示指南 - iOS Emoji 分析器

## 📱 如何在 Xcode 中打开模拟器展示

### 方法一：使用 Xcode 快速启动（推荐）

1. **打开项目**
   - 在 Finder 中找到 `IOS_emoji_analyser.xcodeproj`
   - 双击打开项目

2. **选择模拟器**
   - 在 Xcode 顶部工具栏，点击设备选择器（左上角，Play 按钮旁边）
   - 选择 **iPhone 15** 或 **iPhone 15 Pro**（推荐，性能好）
   - 也可以选择 **iPhone 14** 或其他设备

3. **运行项目**
   - 点击左上角的 **▶️ Play 按钮**，或按 **⌘ + R**
   - 等待构建完成（首次运行可能需要 1-2 分钟）
   - 模拟器会自动启动并运行 App

### 方法二：预先启动模拟器

1. **打开 Simulator 应用**
   - 在 Xcode 菜单栏选择：**Xcode → Open Developer Tool → Simulator**
   - 或使用 Spotlight 搜索 "Simulator"

2. **选择设备**
   - 在 Simulator 菜单栏选择：**File → Open Simulator → iOS 17.x → iPhone 15**

3. **在 Xcode 中运行**
   - 回到 Xcode，确保顶部设备选择器显示的是已启动的模拟器
   - 点击运行按钮

## 🎯 展示内容和流程

### 1️⃣ 项目介绍（1分钟）
- **项目名称**：iOS Emoji 分析器
- **核心功能**：通过语音识别实时分析情感，推荐合适的 emoji 表情
- **技术栈**：SwiftUI + CoreML + Speech Recognition + 自然语言处理

### 2️⃣ 核心功能展示（3-4分钟）

#### 功能 1：语音识别
1. 启动 App 后，点击 **麦克风图标** 开始录音
2. 说一些带有情感的话，例如：
   - "今天真是太开心了！"（快乐情绪）
   - "这个作业好难啊，我有点沮丧"（悲伤情绪）
   - "哇，这个项目做得真棒！"（惊喜情绪）
3. 展示实时文字识别效果

#### 功能 2：情感分析
- 系统会自动分析识别出的文字
- 显示检测到的**情绪类型**（快乐、悲伤、愤怒、惊讶、恐惧、中性）
- 展示**情绪强度**百分比

#### 功能 3：Emoji 推荐
- 根据情感分析结果，推荐相关的 emoji 表情
- 展示 emoji 网格，可点击复制
- 说明推荐算法基于情感-emoji 映射模型

### 3️⃣ 技术亮点（1-2分钟）

#### 🔬 CoreML 模型集成
- 使用预训练的情感分析模型
- 模型文件：`EmojiPredictor_int8.mlpackage`
- 本地推理，保护隐私

#### 🎤 原生 Speech Recognition
- 支持中文实时语音识别
- 使用 iOS 原生 Speech 框架
- 低延迟，高准确度

#### 🎨 SwiftUI 现代化 UI
- 使用最新 SwiftUI 技术
- 响应式设计，流畅动画
- 支持深色模式

## 📋 常见问题处理

### Q1: 模拟器没有声音权限？
**解决方法：**
1. 在模拟器中，打开 **Settings → Privacy & Security → Microphone**
2. 找到 `IOS_emoji_analyser`，确保开关是开启状态
3. 重启 App

### Q2: 语音识别不工作？
**解决方法：**
1. 确保 Mac 系统麦克风权限已授权给 Xcode/Simulator
2. 在 Mac **系统设置 → 隐私与安全性 → 麦克风**，勾选 Simulator
3. 检查 Mac 麦克风是否正常工作

### Q3: 模拟器启动慢？
**解决方法：**
- 首次启动需要加载，耐心等待
- 后续运行会快很多
- 可以提前启动模拟器

### Q4: 编译错误？
**解决方法：**
- 按 **⌘ + Shift + K** 清理项目
- 按 **⌘ + B** 重新构建
- 检查是否所有依赖文件都在项目中
